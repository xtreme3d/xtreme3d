Xtreme3D is a 3D engine for the popular game creation tool GameMaker (former GameMaker: Studio 2). The engine is designed as an external DLL with a set of functions for creating, manipulating and rendering various three-dimensional objects. With Xtreme3D, you can render 3D graphics of any complexity in the GM window. Xtreme3D supports many modern rendering technologies, allows you to create games with high-quality graphics and is currently the only actively developed 3D engine for GameMaker.

This guide was made to help beginners by providing helpful materials: step-by-step tutorials with code samples and a list of all functions with detailed descriptions. We hope that it will make your acquaintance with the engine easier, more interesting and exciting.

Keep in mind that the English version of the help file was translated from Russian using automatic translator, so apologies for any mistakes and inaccuracies that may be here. We are working towards improving the documentation.
