Proxy

Класс: TGLProxyObject, TGLActorProxy, TGLMultiProxy

Прокси - это специальный тип объекта, который не имеет собственной графики для отрисовки - он рисует графику другого объекта, но использует для этого собственную трансформацию. Иными словами, прокси - это лучший способ создать несколько копий 3D-модели, не дублируя ее в памяти.

Мультипрокси - особый подвид прокси, который может рисовать разные объекты в зависимости от удаленности от камеры. С его помощью вы можете реализовать дискретный LOD (level of detail) - систему адаптивной детализации: чем дальше прокси от камеры, тем более низкополигональную модель он рендерит.

Для Актеров следует использовать специальный вид прокси - ActorProxyObject. Он позволяет рендерить копию Актера со своей анимацией - это необходимо, если у вас в игре сосуществуют несколько однотипных персонажей (например, врагов).

ProxyObjectCreate

real = ProxyObjectCreate( object,parent as real );
Создает новый прокси с графикой заданного объекта.
object - id объекта
parent - id родителя.

ProxyObjectSetOptions

real = ProxyObjectSetOptions( proxy,copyeffects,copyobjects,copytransform as real );
Задает опции прокси.
proxy - id прокси
copyeffects - указывает, копировать ли эффекты заданного объекта (речь идет о системах частиц, прикрепленных к объекту)
copyobjects - указывает, копировать ли потомков объекта
copytransform - указывает, копировать ли локальную трансформацию объекта.

ProxyObjectSetTarget

real = ProxyObjectSetTarget( proxy,object as real );
Задает целевой объект прокси.
proxy - id прокси
object - id объекта.

ActorProxyObjectCreate

real = ActorProxyObjectCreate( actor,parent as real );
Создает новый прокси-Актер.
actor - id Актера
parent - id родителя.

ActorProxyObjectSwitchToAnimation

real = ActorProxyObjectSwitchToAnimation( proxy,anim as real );
Переключает анимационную последовательность для прокси-Актера с указанием номера анимации.
proxy - id прокси
anim - порядковый номер анимации.

ActorProxyObjectSetAnimationRange

real = ActorProxyObjectSetAnimationRange( proxy,start,end as real );
Переключает анимационную последовательность для прокси-Актера с указанием первого и последнего кадра.
proxy - id прокси
start,end - первый и последний кадр анимации.

ActorProxyObjectSetInterval

real = ActorProxyObjectSetInterval( proxy,interval as real );
Задает интервал прокси-Актера - время в миллисекундах между двумя ключевыми кадрами. Значение по умолчанию - 100.
proxy - id прокси
interval - интервал.

MultiProxyObjectCreate

real = MultiProxyObjectCreate( parent as real );
Создает новый мультипрокси.
parent - id родителя.

MultiProxyObjectAddTarget

real = MultiProxyObjectAddTarget( mproxy,object,mindist,maxdist as real );
Добавляет целевой объект в мультипрокси и задает диапазон расстояний от камеры, на котором этот объект должен рендериться.
parent - id мультипрокси
object - id объекта
mindist - минимальное расстояние, на котором мультипрокси должен рисовать данный целевой объект
maxdist - максимальное расстояние, на котором мультипрокси должен рисовать данный целевой объект.
