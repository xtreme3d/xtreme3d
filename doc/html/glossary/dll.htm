DLL

Dynamic-Link Library | Динамически подключаемая библиотека
Динамическая библиотека, позволяющая многократное применение различными программными приложениями в среде операционных систем Microsoft Windows и IBM OS/2. В UNIX аналогичные функции выполняют т. н. shared objects («разделяемые объекты»).
Формат файлов DLL придерживается тех же соглашений, что и формат исполняемых файлов, сочетая код, таблицы и ресурсы.
