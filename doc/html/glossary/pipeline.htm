Pipeline | Конвейер
Серия шагов по созданию и отображению трехмерной графики. Первый шаг - трансформация: создается трехмерный объект и отображается на плоскость. Второй шаг - добавление освещенности объекту. Третий шаг - рендеринг цветов и теней многоугольников для соответствующих текстур.
