API

Application Programming Interface | Интерфейс прикладного программирования
Интерфейс между программой и оборудованием. Основной выигрыш от использования API - переносимость (то есть возможность написания одного и того же программного обеспечения для разных аппаратных платформ), стабильность (способность программы быть совместимой с будущими версиями аппаратуры) и простота.

См. также DirectX, OpenGL.
