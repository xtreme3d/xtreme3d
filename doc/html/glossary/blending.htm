Blending | Смешивание
Метод смешивания цветов текущего пикселя и пикселя, находящегося уже в буфере кадра, для получения выходного пикселя. Введем следующие обозначения:
R1, G1, B1, A1 - соответственно представляют красную, зеленую, синюю и альфа компоненты текущего пикселя.
R2, G2, B2, A1 - аналогично для пикселя, уже находящегося в буфере.
Примеры смешивания:
Alpha Blending - добавление прозрачности, т.е. (R1*A1+R2*(1-A1), G1*A1+G2*(1-A1), B1*A1+B2*(1-A1))
Additive Blending - суммирование цветов, т.е. (R1+R2, G1+G2, B1+B2)
Modulate Blending - модуляция цветов, т.е. (R1*R2, G1*G2, B1*B2)

См. также Alpha, Color, Color Model, Frame Buffer.
