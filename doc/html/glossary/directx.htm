DirectX
Набор API для программирования игр в среде MS Windows. Включают в себя средства прорисовки двумерных (DirectDraw) и трехмерных (Direct3D) графических примитивов, определения команд устройств ввода - клавиатуры, мыши, джойстика (DirectInput), работы со звуком (DirectSound), и анимации и трансформации трехмерных объектов. Проще говоря, DirectX - это набор простейших команд, созданных специально для операционной системы Windows, с помощью которых пишут игры. Хотя у DirectX существуют и другие области применения.
Первая версия DirectX вышла в 1994 г.
Основным конкурентом Direct3D является OpenGL.

См. также API, OpenGL.
