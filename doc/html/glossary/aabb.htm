AABB

Axis Aligned Bounding Box | Ограничивающий параллелепипед, выровненный по координатным осям
Ограниченная область в пространстве, окружающая некий объект, в виде прямоугольного параллелепипеда со сторонами, параллельными осям координат в мировой системе. При вращении объекта AABB изменяет свои размеры, но всегда остается ориентированным по осям координат. Второй вариант AABB - использовать в качестве Bounding Box'а куб со сторонами равными максимальному размеру объекта. Таким образом, в любом положении объект не выйдет за пределы AABB, хотя, конечно, при этом теряется точность.
Используется для обнаружения столкновений.

См. также Collision Detection.
