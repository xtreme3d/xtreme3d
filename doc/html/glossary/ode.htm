ODE

Open Dynamics Engine
Бесплатная открытая библиотека промышленного качества для моделирования физики твердых тел. Подходит для симуляции транспортных средств, существ с ногами и движущихся объектов. ODE быстрая, гибкая и надёжная, имеет встроенную систему определения столкновений. ODE разрабатывается Расселом Смитом совместно с командой добровольцев.
