Polygon

Класс: TGLPolygon

ВНИМАНИЕ: страница находится в разработке. Информация ниже была написана для Xtreme3D v2 и может не соответствовать Xtreme3D v3.

Полигон (polygon) - плоскость, сформированная тремя точками в пространстве. Xtreme3D располагает довольно ограниченными средствами отрисовки полигонов, поэтому составить из них сложную модель вряд ли удастся. Хотя полигоны все равно можно использовать, например, для создания различных спецэффектов.

PolygonCreate

real = PolygonCreate( parent as real );
Создает полигон и возвращает его id.
parent - id родителя для полигона.

PolygonAddVertex

real = PolygonAddVertex( polygon,x,y,z as real );
Добавляет вершину (точку) в полигон и возвращает ее id.
polygon - id полигона
x,y,z - координаты вершины.

PolygonSetVertexPosition

real = PolygonSetVertexPosition( polygon,vertex,x,y,z as real );
Задает координаты вершины.
polygon - id полигона
vertex - id вершины
x,y,z - координаты.

PolygonDeleteVertex

real = PolygonDeleteVertex( polygon,vertex as real );
Удаляет вершину из полигона.
polygon - id полигона
vertex - id вершины.
