Primitives

Класс: TGLPlane, TGLCube, TGLSphere, TGLCylinder, TGLCone, TGLAnnulus, TGLTorus, TGLDisk

ВНИМАНИЕ: страница находится в разработке. Информация ниже была написана для Xtreme3D v2 и может не соответствовать Xtreme3D v3.

Примитивы (Primitive) - простейшие геометрические тела, которые генерируются движком. К примитивам относятся плоскость (plane), куб (cube), сфера (sphere), цилиндр (cylinder), конус (cone), полый цилиндр (annulus), тороид (torus) и диск (disk).

PlaneCreate

real = PlaneCreate( singlequad,width,height,xtiles,ytiles,parent as real);
Создает плоскость и возвращает ее id.
singlequad - определяет, разбивать ли плоскость на сетку, или представить ее одним квадратом - false или true (0 и 1 соответственно). При использовании освещения, как правило, лучше разбить плоскость.
width,height - ширина и высота плоскости
xtiles,ytiles - при singlequad = 0 задают количество ячеек сетки по горизонтали и вертикали
parent - id родителя для плоскости (0 - отсутствие родителя).

CubeCreate

real = CubeCreate( width,height,depth,parent as real);
Создает куб (строго говоря, прямоугольный параллелепипед) и возвращает его id.
width,height,depth - ширина, высота и долгота куба
parent - id родителя для куба (0 - отсутствие родителя).

CubeSetNormalDirection

real = CubeSetNormalDirection( cube,nd as real );
Задает направление нормалей куба. Доступны следующие значения nd:
ndOutside = 0 - нормали направлены внутрь (значение по умолчанию)
ndInside = 1 - нормали направлены наружу.
cube - id куба
nd - направление нормалей.

SphereCreate

real = SphereCreate( radius,slices,stacks,parent as real);
Создает сферу и возвращает ее id.
radius - радиус сферы
slices,stacks - задают количество меридиан и параллелей
parent - id родителя для сферы (0 - отсутствие родителя).

CylinderCreate

real = CylinderCreate ( topradius,bottomradius,height,slices,stacks,loops,parent as real);
Создает цилиндр и возвращает его id.
topradius - радиус верхнего основания
bottomradius - радиус нижнего основания
height - высота (значение по умолчанию: 1)
slices,stacks,loops - задают количество ребер, сегментов и концентрических окружностей
parent - id родителя для цилиндра (0 - отсутствие родителя).

ConeCreate

real = ConeCreate ( bottomradius,height,slices,stacks,loops,parent as real);
Создает конус и возвращает его id.
bottomradius - радиус основания
height - высота (значение по умолчанию: 1)
slices,stacks,loops - задают количество ребер, сегментов и концентрических окружностей
parent - id родителя для конуса (0 - отсутствие родителя).

AnnulusCreate

real = AnnulusCreate ( innerradius,outerradius,height,slices,stacks,loops,parent as real);
Создает полый цилиндр (кольцо) и возвращает его id.
innerradius - радиус внутренней окружности
outerradius - радиус внешней окружности
height - высота (значение по умолчанию: 1)
slices,stacks,loops - задают количество ребер, сегментов и концентрических окружностей
parent - id родителя для полого цилиндра (0 - отсутствие родителя).

TorusCreate

real = TorusCreate ( outerradius,innerradius,rings,sides,parent as real);
Создает тороид и возвращает его id. Тороид, или тор - это тело, напоминающее бублик.
outerradius - радиус внешней окружности
innerradius - радиус внутренней окружности
rings,sides - задают количество колец и сегментов
parent - id родителя для тороида (0 - отсутствие родителя).

DiskCreate

real = DiskCreate ( innerradius,outerradius,startangle,sweepangle,loops,slices,parent as real);
Создает диск и возвращает его id.
innerradius - радиус внутренней окружности
outerradius - радиус внешней окружности
startangle,sweepangle - начальный и конечный угол сектора. Для замкнутого диска эти значения соответствуют 0, 360
loops,slices - задают количество концентрических окружностей и ребер
parent - id родителя для полого цилиндра (0 - отсутствие родителя).
