ThorFX

Класс: TGLThorFXManager, TThor

ВНИМАНИЕ: страница находится в разработке. Информация ниже была написана для Xtreme3D v2 и может не соответствовать Xtreme3D v3.

ThorFX - это узкоспециализированная система частиц, ориентированная на работу со сплошной линией. При помощи ThorFX можно создать невероятно реалистичные молнии, электрические разряды, лучи лазера, потоки плазмы и прочие эффекты, в которых используется прямая или ломаная линия.
Система ThorFX состоит из двух компонентов: стержень, или ядро (core) и свечение (glow). Стержень - сплошная линия, закрашенная одним цветом, а свечение представляет собой совокупность отдельных частиц (спрайтов), равномерно распределенных по всей длине стержня. При достаточном количестве частиц они визуально сольются в сплошной объем, и будет казаться, что стержень светится, как неоновая трубка.
Для воссоздания эффекта электрического разряда (его еще называют эффектом Тесла, по имени выдающегося физика Николы Тесла) включается электромагнитная вибрация и смещение - и перед нами уже не неоновая трубка, а настоящая молния!
Положение системы в пространстве задается ее собственными координатами, а также координатами целевой точки, в которую должна "бить" молния. Вместо точки можно указать целевой объект, и тогда молния будет "бить" в него.

В некоторых случаях необходимо создать сразу несколько одинаковых эффектов. Не очень-то удобно повторять все настройки для каждого из них. Специально для такой ситуации предусмотрен менеджер ThorFX. Менеджер - это что-то вроде сервера, на котором выполняются все вычисления, связанные с эффектом. А отдельные системы ThorFX - это клиенты, использующие настройки указанного для них менеджера. Вы просто создаете менеджер, настраиваете его как вам угодно, а затем добавляете любое количество систем в любых нужных вам местах сцены. Изменения, внесенные в настройки менеджера, автоматически коснутся всех подчиненных ему систем.

ThorFXManagerCreate

real = ThorFXManagerCreate();
Создает новый менеджер ThorFX и возвращает его id.

ThorFXCreate

real = ThorFXCreate( manager,parent as real );
Создает новую систему ThorFX и возвращает ее id.
manager - id менеджера ThorFX
parent - id родителя для системы.

ThorFXSetColor

real = ThorFXSetColor( manager,incolor,inalpha,outcolor,outalpha,ccolor,calpha as real );
Задает цвет системы.
manager - id менеджера ThorFX
incolor,inalpha - цвет и прозрачность центра свечения
outcolor,outalpha - цвет и прозрачность периферии свечения
ccolor,calpha - цвет и прозрачность стержня.

ThorFXEnableCore

real = ThorFXEnableCore( manager,mode as real );
Включает или выключает стержень.
manager - id менеджера ThorFX
mode - true или false (1 и 0 соответственно).

ThorFXEnableGlow

real = ThorFXEnableGlow( manager,mode as real );
Включает или выключает свечение.
manager - id менеджера ThorFX
mode - true или false (1 и 0 соответственно).

ThorFXSetMaxParticles

real = ThorFXSetMaxParticles( manager,particles as real );
Задает количество частиц в свечении. Чем выше это число, тем реалистичнее выглядит свечение.
manager - id менеджера ThorFX
particles - количество частиц (от 0 до 255).

ThorFXSetGlowSize

real = ThorFXSetGlowSize( manager,size as real );
Задает размер свечения.
manager - id менеджера ThorFX
size - размер.

ThorFXSetVibrate

real = ThorFXSetVibrate( manager,vibrate as real );
Задает уровень вибрации стержня (амплитуду электромагнитных колебаний). Ноль означает отсутствие вибрации.
manager - id менеджера ThorFX
vibrate - уровень вибрации.

ThorFXSetWildness

real = ThorFXSetWildness( manager,wildness as real );
Задает уровень смещения стержня относительно осевой линии. Электрический разряд как бы "дергается", усиливая впечатление от эффекта. Такое часто можно наблюдать, как в природе, так и в научных экспериментах.
Ноль означает отсутствие смещения.
manager - id менеджера ThorFX
vibrate - уровень вибрации.

ThorFXSetTarget

real = ThorFXSetTarget( manager,x,y,z as real );
Задает целевую точку системы. Координаты этой точки всегда задаются локально, то есть относительно собственных координат системы. Если бы координаты задавались абсолютно, все созданные системы ThorFX были направлены в одну и ту же точку, что, согласитесь, обычно совсем нежелательно.
manager - id менеджера ThorFX
x,y,z - координаты точки.

ThorFXSetTargetAsObject

real = ThorFXSetTargetAsObject( manager,object as real );
Задает целевой объект системы. В этом случае все созданные системы ThorFX будут направлены в один и тот же объект. Это может быть полезно, скажем, когда вы создаете специальное электрическое оружие, которое бьет в ближайший к нему объект (такое оружие можно встретить, например, в GTA 2).
manager - id менеджера ThorFX
object - id объекта.
