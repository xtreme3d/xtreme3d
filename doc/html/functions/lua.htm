Lua

Класс: TLua

Начиная с версии 3.6, Xtreme3D включает поддержку Lua - популярного скриптового языка программирования. Для Lua-скриптов доступны все функции Xtreme3D. Вы можете использовать эту возможность для того, чтобы реализовать динамическую логику. Например, поведение игровых объектов может задаваться скриптами, что позволяет расширять игру, создавать моды. На самом деле, практически вся игра может быть целиком написана на Lua.
Для использования Lua необходимо поместить lua.dll в каталог с проектом.
Вы можете узнать подробнее о языке на сайте https://www.lua.org.

LuaManagerCreate

real = LuaManagerCreate( );
Создает менеджер Lua и возвращает его id.

LuaManagerRunScript

real = LuaManagerRunScript( lua as real, script as string );
Выполняет скрипт Lua.
lua - id менеджера Lua
script - строка, содержащая скрипт.

LuaManagerSetConstantNumber

real = LuaManagerSetConstantNumber( lua as real, name as string, value as real );
Задает числовую константу - то есть, неизменяемое значение, доступное в скриптах по заданному имени. Числовую константу можно использовать не только для чисел как таковых, но и для id'ов объектов Xtreme3D - вы можете создавать объекты в основной программе, а управлять ими в скриптах.
lua - id менеджера Lua
name - имя константы
value - значение константы.

LuaManagerSetConstantString

real = LuaManagerSetConstantString( lua as real, name,value as string );
Задает строковую константу.
lua - id менеджера Lua
name - имя константы
value - значение константы.
