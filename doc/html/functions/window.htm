Window

Класс: TForm

Функции для создания дополнительных окон и управления ими. Вы можете рендерить в созданном окне графику Xtreme3D, если передадите при создании вида соответствующий идентификатор (HWND) при помощи функции WindowGetHandle. Обратите внимание, что перед завершением игры все созданные окна нужно удалить функцией WindowDestroy, иначе возникнет ошибка.

WindowCreate

real = WindowCreate( x,y,width,height,resizeable as real );
Создает окно и возвращает его id.
x,y - позиция окна на экране
width,height - ширина и высота окна
resizeable - может ли пользователь изменять размер окна (true или false). Обратите внимание, что изменение размера окна не приводит к изменению размеров подключенного к нему вида, это нужно делать вручную.

WindowCenter

real = WindowCenter( window as real );
Помещает окно в центр экрана.
window - id окна.

WindowResize

real = WindowResize( window,x,y,width,height as real );
Задает позицию и размер окна. Обратите внимание, что изменение размера окна не приводит к изменению размеров подключенного к нему вида, это нужно делать вручную.
window - id окна
x,y - позиция окна на экране
width,height - ширина и высота окна.

WindowGetPosition

real = WindowGetPosition( window,index as real );
Возвращает координату позиции окна (левого верхнего угла).
window - id окна
index - если этот параметр равен 0, функция возвращает координату X, если 1 - Y.

WindowGetSize

real = WindowGetSize( window,index as real );
Возвращает ширину или высоту окна.
window - id окна
index - если этот параметр равен 0, функция возвращает ширину, если 1 - высоту.

WindowGetHandle

real = WindowGetHandle( window as real );
Возвращает идентификатор окна (HWND).
window - id окна.

WindowSetTitle

real = WindowSetTitle( window as real, title as string );
Задает заголовок окна.
window - id окна
title - заголовок.

WindowDestroy

real = WindowDestroy( window as real );
Удаляет окно.
window - id окна.
