Blending | Смешивание
Смешивание двух цветов в модели RGB для получения третьего. Используется для наложения одного изображения на другое с использованием различных эффектов прозрачности.

Существует множество разных видов смешивания. Введем следующие обозначения:
R1, G1, B1, A1 - соответственно представляют красную, зеленую, синюю и альфа компоненты первого цвета.
R2, G2, B2, A1 - аналогично для второго цвета.
Примеры смешивания:
Alpha Blending - альфа-смешивание: (R1*A1+R2*(1-A1), G1*A1+G2*(1-A1), B1*A1+B2*(1-A1))
Additive Blending - суммирование: (R1+R2, G1+G2, B1+B2)
Modulate Blending - умножение: (R1*R2, G1*G2, B1*B2)

См. также Alpha, Color, Color Model.
