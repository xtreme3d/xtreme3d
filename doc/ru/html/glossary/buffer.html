Buffer

Buffer | Буфер
Область временного хранения данных в памяти. Обычно буфер используется для обмена данными с периферией с высокой задержкой (видео, аудио, принтер, пишущий дисковод и т.д.). Некоторые буферы являются частью адресуемой системной памяти, а некоторые являются частью самих устройств.

См. также Frame Buffer, Back Buffer, Front Buffer, Line Buffer, Double Buffering.
