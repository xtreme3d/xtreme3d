Back Buffer

Back Buffer | Вторичный буфер
Область видеопамяти, в которую рендерятся объекты трехмерной сцены. Вывод изображения на экран осуществляется через Front Buffer (первичный буфер). По завершении рендеринга кадра, буферы меняются местами (buffer swapping). Таким образом, зритель всегда видит на экране готовое изображение.

См. также Buffer, Frame Buffer, Front Buffer, Double Buffering.
