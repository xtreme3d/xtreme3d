Text

Класс: -

Функции для работы со строками и текстами.

TextRead

string = TextRead( filename as string );
Загружает текстовый файл в строку и возвращает ее.
filename - имя файла.

TextConvertANSIToUTF8

string = TextConvertANSIToUTF8( text as string );
Конвертирует строку из кодировки ANSI в UTF-8 (Юникод). Входная строка не модифицируется, функция создает в памяти новую строку и возвращает ее. Во избежание утечки памяти не рекомендуется вызывать эту функцию постоянно - сконвертируйте строку один раз в начале программы и используйте ее.
text - текст.
