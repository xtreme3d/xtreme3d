Trail

Класс: TGLTrail

Объект trail ("след") создает позади другого объекта красивый шлейф, возникающий при быстром движении. Такой эффект можно использовать, например, в авиасимуляторах.

TrailCreate

real = TrailCreate( object,parent as real );
Создает новый объект trail и возвращает его id.
object - id целевого объекта, к которому должен быть применен эффект шлейфа
parent - id родителя.

TrailSetObject

real = TrailSetObject( trail,object as real );
Меняет целевой объект, к которому должен быть применен эффект шлейфа
trail - id объекта trail
object - id нового целевого объекта.

TrailSetAlpha

real = TrailSetAlpha( trail,alpha,fade as real );
Задает прозрачность шлейфа.
trail - id объекта trail
alpha - прозрачность
fade - определяет, нужно ли сделать шлейф плавно исчезающим: true или false (1 и 0 соответственно).

TrailSetLimits

real = TrailSetLimits( trail,vl,tl as real );
Задает ограничения шлейфа.
trail - id объекта trail
vl - максимальная длина шлейфа
tl - время до исчезновения шлейфа (в секундах).

TrailSetMinDistance

real = TrailSetMinDistance( trail,distance as real );
Задает расстояние, при прохождении которого целевым объектом следует создать очередной сегмент шлейфа. Чем выше это значение, тем качественнее эффект.
trail - id объекта trail
distance - расстояние.

TrailSetUVScale

real = TrailSetUVScale( trail,uv as real );
Сведения отсутствуют.
trail - id объекта trail
uv - сведения отсутствуют.

TrailSetMarkStyle

real = TrailSetMarkStyle( trail,ms as real );
Задает стиль шлейфа (ms). Доступны следующие значения ms:
msUp = 0 - шлейф всегда выровнен по вертикали
msDirection = 1 - шлейф похож на экран радара: движется по кругу относительно локального центра объекта trail, следуя за целевым объектом
msFaceCamera = 2 - шлейф всегда выровнен по камере и поэтому кажется объемным (значение по умолчанию).
trail - id объекта trail
ms - стиль шлейфа.

TrailSetMarkWidth

real = TrailSetMarkWidth( trail,width as real );
Задает размер шлейфа.
trail - id объекта trail
width - размер (по умолчанию: 1).

TrailSetEnabled

real = TrailSetEnabled( trail,mode as real );
Включает или выключает шлейф.
trail - id объекта trail
mode - true или false (1 и 0 соответственно).

TrailClearMarks

real = TrailClearMarks( trail as real );
Сбрасывает состояние шлейфа. Полезно, когда нужно резко остановить объект, не дожидаясь, когда шлейф сам исчезнет.
trail - id объекта trail.
