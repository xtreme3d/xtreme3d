Dummycube

Класс: TGLDummyCube

Манекен (Dummycube) - это специальный объект, невидимый, но имеющий позицию и ориентацию в пространстве. Обработка манекенов осуществляется очень быстро, и, так как они, подобно другим объектам, могут иметь родителей и потомков, это идеальные объекты для создания групп и иерархий. Свойства манекенов и систем, построенных на их основе, применяются в таких областях, как механизмы движения объектов, шарнирные конструкции, наложение эффектов и проверка столкновений.

DummycubeCreate

real = DummycubeCreate( parent as real );
Создает новый манекен и возвращает его id.
parent - id родителя для манекена (0 - отсутствие родителя).

DummycubeAmalgamate

real = DummycubeAmalgamate ( object,mode as real );
Соединяет всех потомков манекена в один объект OpenGL. Это активирует особый режим рендеринга, при котором все потомки манекена будут занесены в один список отображения. Таким образом, в некоторых ситуациях можно достигнуть заметного прироста производительности, но все изменения, коснувшиеся потомков, будут игнорироваться. Некоторые объекты, имеющие собственное управление списком отображения, могут быть несовместимы с DummycubeAmalgamate. Кроме того, этот режим не позволяет осуществлять сортировку и отбор видимости обычным образом. Короче говоря, эта опция хороша для статичной непрозрачной геометрии, или когда положение камеры не меняется в течение длительного времени.
object - id манекена
mode - true или false (1 и 0 соответственно).

DummycubeSetCameraMode

real = DummycubeSetCameraMode( object,cim as real );
Позволяет зафиксировать позицию (cimPosition) или ориентацию (cimOrientation) манекена и его потомков по отношению к активной камере. То есть, в первом случае невозможно отойти или приблизиться к объекту, во втором - убрать объект из вида.
object - id манекена
cim - cimNone (значение по умолчанию), cimPosition или cimOrientation (0, 1 или 2 соответственно).

DummycubeSetVisible

real = DummycubeSetVisible( object,mode as real );
Переключает видимость манекена. Манекен отображается как каркасный куб.
object - id манекена
mode - true или false (1 и 0 соответственно).

DummycubeSetEdgeColor

real = DummycubeSetEdgeColor( object,color as real );
Задает цвет ребер куба, в виде которого отображается видимый манекен.
object - id манекена
color - цвет.

DummycubeSetCubeSize

real = DummycubeSetCubeSize( object,size as real );
Задает размер грани куба, в виде которого отображается видимый манекен. Размер по умолчанию - 1.
object - id манекена
size - размер.
