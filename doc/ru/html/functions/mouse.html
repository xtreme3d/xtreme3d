Mouse

Класс: -

ВНИМАНИЕ: страница находится в разработке. Информация ниже была написана для Xtreme3D v2 и может не соответствовать Xtreme3D v3.

Простые функции управления мышью.

MouseGetX

real = MouseGetX();
Возвращает координату X курсора мыши относительно левого верхнего угла экрана.
Функция полностью аналогична display_mouse_get_x.

MouseGetY

real = MouseGetY();
Возвращает координату Y курсора мыши относительно левого верхнего угла экрана.
Функция полностью аналогична display_mouse_get_y.

MouseSetPosition

real = MouseSetPosition( x,y as real );
Задает координаты курсора мыши относительно левого верхнего угла экрана.
Функция полностью аналогична display_mouse_set.
x,y - координаты.

MouseShowCursor

real = MouseShowCursor( mode as real );
Включает или выключает отображение курсора мыши.
mode - true или false.
