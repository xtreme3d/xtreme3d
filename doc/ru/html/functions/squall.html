Squall

Класс: -

Squall - это компактный и мощный звуковой движок, разработанный Маратом Гилязетдиновым (студия AntiTank). В российском геймдеве Squall был очень популярен в "нулевых" и конкурировал с BASS и FMOD. В 2009 году движок был открыт, однако автор прекратил его разработку, поэтому мы из чувства ностальгии решили сделать его частью проекта Xtreme3D. Несмотря на отсутствие развития, Squall все еще остается отличным решением для воспроизведения звука в играх под Windows - ему мало серьезных альтернатив среди свободного ПО. Движок надежен, многократно проверен на практике, предоставляет удобный, продуманный API. Squall читает файлы WAV, MP3, OGG/Vorbis, воспроизводит звук в отдельном потоке через DirectSound (требуется DirectX 7 или выше), поддерживает 3D-звук и аппаратные эффекты HRTF (CreativeLab EAX, Sensaura ZoomFX).
Архитектура Squall основана на разделении звуковых данных от собственно источников звука - то есть, можно загрузить звуковой файл один раз, а затем воспроизводить в разные моменты времени и с разными настройками. Источники звука называются каналами (channel). Канал может располагаться в трехмерном пространстве - в этом случае для него задаются позиция, скорость, параметры EAX и ZoomFX. Также каналам можно индивидуально задавать громкость и частоту.
Для использования Squall необходимо посместить squall.dll в каталог с проектом.

SquallInit

real = SquallInit( );
Инициализирует Squall. Эту функцию необходимо вызвать в начале игры перед тем, как использовать остальные.

SquallAddSound

real = SquallAddSound( filename as string );
Создает новый звук из файла и возвращает его id.
filename - имя файла.

SquallPlay

real = SquallPlay( snd,loop as real );
Запускает воспроизведение звука и возвращает id канала воспроизведения.
snd - id звука
loop - зацикливать ли воспроизведение (true или false).

SquallStop

real = SquallStop( chan as real );
Останавливает воспроизведение заданного канала.
chan - id канала.
