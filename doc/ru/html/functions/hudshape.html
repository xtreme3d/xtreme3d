HUDShape

Класс: TGLHUDShape

HUDShape - это двумерная фигура, которая отрисовывается на экране поверх проекции. Xtreme3D поддерживает четыре типа фигур: прямоугольник (rectangle), окружность (circle), отрезок (line) и меш (mesh). Фигуры могут иметь или не иметь материал. Фигурам можно индивидуально задавать цвет и прозрачность независимо от материала.
Фигуры можно использовать для отрисовки различных сложных элементов интерфейса - например, окружность хорошо подходит для создания круговой шкалы здоровья.

HUDShapeRectangleCreate

real = HUDShapeRectangleCreate( width,height,parent as real );
Создает фигуру-прямоугольник и возвращает ее id.
width,height - ширина и высота прямоугольника в пикселях.
parent - id родителя для фигуры (0 - отсутствие родителя).

HUDShapeCircleCreate

real = HUDShapeCircleCreate( radius,slices,startAngle,endAngle,parent as real );
Создает фигуру-окружность и возвращает ее id.
radius - радиус окружности
slices - количество ребер контура
startAngle,endAngle - начальный и конечный углы (в градусах). Эти параметры позволяют отрисовать не целую окружность, а ее сектор. Если задать 0 и 360, будет отрисована целая окружность
parent - id родителя для фигуры (0 - отсутствие родителя).

HUDShapeLineCreate

real = HUDShapeLineCreate( x1,y1,x2,y2,parent as real );
Создает фигуру-отрезок и возвращает ее id.
x1,y1 - начальная точка отрезка в локальных координатах объекта (при собственном размере фигуры 1х1 они соответствуют пикселям)
x2,y2 - конечная точка отрезка в локальных координатах объекта (при собственном размере фигуры 1х1 они соответствуют пикселям)
parent - id родителя для фигуры (0 - отсутствие родителя).

HUDShapeMeshCreate

real = HUDShapeMeshCreate( parent as real );
Создает фигуру-меш и возвращает ее id. Меш позволяет отрисовать произвольную фигуру, составленную из треугольников при помощи функций HUDShapeMeshAddVertex и HUDShapeMeshAddTriangle.
parent - id родителя для фигуры (0 - отсутствие родителя).

HUDShapeSetSize

real = HUDShapeSetSize( shape,width,height as real );
Задает размер фигуры. Обратите внимание, что размер имеет для разных фигур разную трактовку. Для прямоугольников размер соответствует реальному размеру на экране в пикселях. Для окружностей размер также соответствует пикселям и влияет на горизонтальный и вертикальный диаметры (то есть, при использовании неодинаковых ширины и высоты окружность превращается в эллипс). Для отрезков и мешей размер трактуется как коэффициент масштабирования, при этом единичный коэффициент соответствует ширине и высоте, равным 1. При таком размере отрезок и меш имеют исходный масштаб, и координаты точек будут соответствовать пикселям на экране.
shape - id фигуры
width,height - ширина и высота фигуры.

HUDShapeScale

real = HUDShapeScale( shape,hscale,vscale as real );
Изменяет размер фигуры относительно текущего размера.
shape - id фигуры
hscale,vscale - добавляются к текущим значениям ширины и высоты.

HUDShapeSetRotation

real = HUDShapeSetRotation( shape,angle as real );
Задает угол поворота фигуры.
shape - id фигуры
angle - угол поворота (в градусах).

HUDShapeRotate

real = HUDShapeRotate( shape,angle as real );
Изменяет поворот фигуры относительно текущего угла.
shape - id фигуры
angle - добавляется к текущему углу поворота.

HUDShapeSetColor

real = HUDShapeSetColor( shape,color,alpha as real );
Задает цвет и прозрачность фигуры. Этот цвет не зависит от материала, присвоенного фигуре - вы можете не присваивать материал и задавать цвет фигурам индивидуально. Если материал присвоен, то собственный цвет фигуры смешивается с текстурой материала.
shape - id фигуры
color - цвет
alpha - прозрачность.

HUDShapeSetOrigin

real = HUDShapeSetOrigin( shape,x,y as real );
Задает исходную точку фигуры - то есть, точку, вокруг которой осуществляется вращение фигуры. По умолчанию вращение осуществляется вокруг центра фигуры.
shape - id фигуры
x,y - координаты исходной точки в пикселях относительно центра фигуры (координаты (0, 0) соответствуют центру).

HUDShapeCircleSetRadius

real = HUDShapeCircleSetRadius( shape,radius as real );
Задает радиус фигуры-окружности. Вызов этой функции эквивалентен установке ширины и высоты, равных двухкратному радиусу.
shape - id фигуры
radius - радиус.

HUDShapeCircleSetSlices

real = HUDShapeCircleSetSlices( shape,slices as real );
Задает количество ребер контура фигуры-окружности.
shape - id фигуры
slices - количество ребер контура.

HUDShapeCircleSetAngles

real = HUDShapeCircleSetAngles( shape,startAngle,endAngle as real );
Задает начальный и конечный углы фигуры-окружности. Эти параметры позволяют отрисовать не целую окружность, а ее сектор. Если задать 0 и 360, будет отрисована целая окружность
shape - id фигуры
startAngle,endAngle - начальный и конечный углы (в градусах).

HUDShapeLineSetPoints

real = HUDShapeLineSetPoints( shape,x1,y1,x2,y2 as real );
Задает начальную и конечную точки фигуры-отрезка. Точки задаются в локальных координатах объекта, поэтому при изменении размера фигуры они могут не совпадать с координатами пикселей на экране (они соответствуют пикселям при размере 1х1).
shape - id фигуры
x1,y1 - начальная точка отрезка
x2,y2 - конечная точка отрезка.

HUDShapeLineSetWidth

real = HUDShapeLineSetWidth( shape,width as real );
Задает толщину фигуры-отрезка.
shape - id фигуры
width - толщина отрезка в пикселях.

HUDShapeMeshAddVertex

real = HUDShapeMeshAddVertex( shape,x,y,u,v as real );
Добавляет новую вершину в фигуру-меш.
shape - id фигуры
x,y - вершина в локальных координатах объекта
u,v - текстурные координаты вершины.

HUDShapeMeshAddTriangle

real = HUDShapeMeshAddTriangle( shape,v1,v2,v3 as real );
Добавляет новый треугольник в фигуру-меш. Треугольник образуется вершинами индексов v1, v2, v3.
shape - id фигуры
v1,v2,v3 - индексы вершин.

HUDShapeMeshSetVertex

real = HUDShapeMeshSetVertex( shape,index,x,y as real );
Модифицирует существующую вершину фигуры-меша.
shape - id фигуры
index - индекс вершины
x,y - вершина в локальных координатах объекта.

HUDShapeMeshSetTexCoord

real = HUDShapeMeshSetTexCoord( shape,index,u,v as real );
Модифицирует текстурные координаты существующей вершины фигуры-меша.
shape - id фигуры
index - индекс вершины
u,v - текстурные координаты.
