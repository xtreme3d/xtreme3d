Авторы Xtreme3D

Автор идеи, оригинального API и разработчик Xtreme3D до версии 2.0.2.0 включительно - Xception.

Основной разработчик Xtreme3D 3.x-4.x - Gecko.

Разработчик Xtreme3D 3.8, автор функций Pipe, TilePlane, VerletWorld, дополнительных функций Viewer, Light, Grid, Material, Actor, Freeform, HUDSprite, Sprite, FireFX, Lines, Movement - FireRun

Автор функций DCE и Input - Rutraple aka Hacker.

Автор функций Grid - Ghost.

Автор функции WindowSetIcon - gadmaker.

Также выражаем огромную благодарность Майку Лишке, Эрику Гранжу и всем контрибьюторам проекта GLScene за создание столь замечательной свободной графической библиотеки.
