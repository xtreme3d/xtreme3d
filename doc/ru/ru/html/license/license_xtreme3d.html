Проект Xtreme3D включает модифицированный код проекта GLScene, который также доступен по лицензии MPL. Проект также включает код сторонних библиотек, на которые не распространяется лицензия MPL:

  • Simple Dictionary
  • Hash Library от Ciaran McCreesh
  • Декодер файлов формата LOD
  • Модифицированная привязка FreeType от проекта Anti-Grain Geometry
  • Физический движок Kraft от Benjamin Rosseaux.
