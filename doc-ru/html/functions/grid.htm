Grid

Класс: TGLXYZGrid

Сетка (grid) - это набор пересекающихся отрезков, образующих равномерную прямоугольную сетку, двумерную или трехмерную. Этот объект удобен для использования в отладочных и вспомогательных целях, когда нужно визуализировать координатную плоскость.

GridCreate

real = GridCreate( x,y,z,step,parent as real );
Создает новую сетку и возвращает еe id.
x,y,z - полуразмер сетки по трем осям
step - размер ячейки сетки
parent - id родителя.

GridSetLineStyle

real = GridSetLineStyle( grid,gls as real );
Задает стиль линий сетки: glsSegments (сведения отсутствуют) или glsLine (сведения отсутствуют).
grid - id сетки
gls - стиль линий сетки: glsSegments = 0 или glsLine = 1.

GridSetLineSmoothing

real = GridSetLineSmoothing( grid,mode as real );
Переключает сглаживание линий сетки.
grid - id сетки
mode - сглаживание: true или false (1 и 0 соответственно).

GridSetParts

real = GridSetParts( grid,gp as real );
Задает координатную плоскость сетки.
grid - id сетки
gp - координатная плоскость. Доступны следующие значения gp:
gpXY = 0 - плоскость XY
gpYZ = 1 - плоскость YZ
gpXZ = 2 - плоскость XZ
gpXYZ = 3 - в этом режиме будет отрисована трехмерная сетка (XYZ)

GridSetColor

real = GridSetColor( grid,color,alpha as real );
Задает цвет сетки.
grid - id сетки
color - цвет
alpha - прозрачность.

GridSetSize

real = GridSetSize( grid,size as real );
Задает толщину линии сетки.
grid - id сетки
size - толщина линии.

GridSetPattern

real = GridSetPattern( grid,pattern as real );
Сведения отсутствуют.
grid - id сетки
pattern - сведения отсутствуют.

GridSetTile

real = GridSetTile( grid,x,y,z as real );
Сведения отсутствуют.
grid - id сетки
x,y,z - сведения отсутствуют.

GridSetStep

real = GridSetStep( grid,step as real );
Сведения отсутствуют.
grid - id сетки
step - сведения отсутствуют.
