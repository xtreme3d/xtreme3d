Primitives

Класс: TGLPlane, TGLCube, TGLSphere, TGLCylinder, TGLCone, TGLAnnulus, TGLTorus, TGLDisk, TGLFrustrum, TGLDodecahedron, TGLIcosahedron, TGLTeapot

Примитивы (Primitive) - простейшие геометрические тела, которые генерируются движком. К примитивам относятся плоскость (plane), куб (cube), сфера (sphere), цилиндр (cylinder), конус (cone), полый цилиндр (annulus), тор (torus), диск (disk), усеченная пирамида (frustrum), додекаэдр (dodecahedron), икосаэдр (icosahedron) и чайник Юта (teapot).

PlaneCreate

real = PlaneCreate( singlequad,width,height,xtiles,ytiles,parent as real);
Создает плоскость и возвращает ее id.
singlequad - определяет, разбивать ли плоскость на сетку, или представить ее одним квадратом - false или true (0 и 1 соответственно). При использовании освещения, как правило, лучше разбить плоскость
width,height - ширина и высота плоскости
xtiles,ytiles - при singlequad = 0 задают количество ячеек сетки по горизонтали и вертикали
parent - id родителя для плоскости (0 - отсутствие родителя).

PlaneSetOptions

real = PlaneSetOptions( plane,singlequad,xtiles,ytiles as real);
Задает параметры плоскости.
plane - id плоскости
singlequad - определяет, разбивать ли плоскость на сетку, или представить ее одним квадратом - false или true (0 и 1 соответственно). При использовании освещения, как правило, лучше разбить плоскость
xtiles,ytiles - при singlequad = 0 задают количество ячеек сетки по горизонтали и вертикали.

PlaneGetOptions

real = PlaneGetOptions( plane,ind as real);
Возвращает параметры плоскости.
plane - id плоскости
ind - индекс параметра (0 = xtiles, 1 = ytiles).

CubeCreate

real = CubeCreate( width,height,depth,parent as real);
Создает куб (строго говоря, прямоугольный параллелепипед) и возвращает его id.
width,height,depth - ширина, высота и долгота куба
parent - id родителя для куба (0 - отсутствие родителя).

CubeSetNormalDirection

real = CubeSetNormalDirection( cube,nd as real );
Задает направление нормалей куба. Доступны следующие значения nd:
ndOutside = 0 - нормали направлены внутрь (значение по умолчанию)
ndInside = 1 - нормали направлены наружу.
cube - id куба
nd - направление нормалей.

CubeGetNormalDirection

real = CubeGetNormalDirection( cube as real );
Возвращает направление нормалей куба.
cube - id куба.

SphereCreate

real = SphereCreate( radius,slices,stacks,parent as real);
Создает сферу и возвращает ее id.
radius - радиус сферы
slices,stacks - задают количество меридиан и параллелей
parent - id родителя для сферы (0 - отсутствие родителя).

SphereSetOptions

real = SphereSetOptions( sphere,radius,slices,stacks as real);
Задает параметры сферы.
sphere - id сферы
radius - радиус сферы
slices,stacks - задают количество меридиан и параллелей.

SphereGetOptions

real = SphereGetOptions( sphere,ind as real);
Возвращает параметры сферы.
sphere - id сферы
ind - индекс параметра (0 = radius, 1 = slices, 2 = stacks).

SphereSetAngleLimits

Сведения отсутствуют.
sphere - id сферы
start,stop - cведения отсутствуют
top,bottom - cведения отсутствуют.

SphereGetAngleLimits

real = SphereGetAngleLimits( sphere,ind as real);
Сведения отсутствуют.
sphere - id сферы
ind - индекс параметра (0 = start, 1 = stop, 2 = top, 3 = bottom).

CylinderCreate

real = CylinderCreate ( topradius,bottomradius,height,slices,stacks,loops,parent as real);
Создает цилиндр и возвращает его id.
topradius - радиус верхнего основания
bottomradius - радиус нижнего основания
height - высота (значение по умолчанию: 1)
slices,stacks,loops - задают количество ребер, сегментов и концентрических окружностей
parent - id родителя для цилиндра (0 - отсутствие родителя).

ConeCreate

real = ConeCreate ( bottomradius,height,slices,stacks,loops,parent as real);
Создает конус и возвращает его id.
bottomradius - радиус основания
height - высота (значение по умолчанию: 1)
slices,stacks,loops - задают количество ребер, сегментов и концентрических окружностей
parent - id родителя для конуса (0 - отсутствие родителя).

AnnulusCreate

real = AnnulusCreate ( innerradius,outerradius,height,slices,stacks,loops,parent as real);
Создает полый цилиндр (кольцо) и возвращает его id.
innerradius - радиус внутренней окружности
outerradius - радиус внешней окружности
height - высота (значение по умолчанию: 1)
slices,stacks,loops - задают количество ребер, сегментов и концентрических окружностей
parent - id родителя для полого цилиндра (0 - отсутствие родителя).

TorusCreate

real = TorusCreate ( innerradius,outerradiusrings,rings,sides,parent as real);
Создает тор и возвращает его id. Тор - это тело, напоминающее бублик.
innerradius - радиус внутренней окружности
outerradius - радиус внешней окружности
rings,sides - задают количество колец и сегментов
parent - id родителя для тороида (0 - отсутствие родителя).

DiskCreate

real = DiskCreate ( innerradius,outerradius,startangle,sweepangle,loops,slices,parent as real);
Создает диск и возвращает его id.
innerradius - радиус внутренней окружности
outerradius - радиус внешней окружности
startangle,sweepangle - начальный и конечный угол сектора. Для замкнутого диска эти значения соответствуют 0, 360
loops,slices - задают количество концентрических окружностей и ребер
parent - id родителя для полого цилиндра (0 - отсутствие родителя).

FrustrumCreate

real = FrustrumCreate ( basewidth,basedepth,apexheight,cutheight,parent as real);
Создает усеченную пирамиду и возвращает ее id.
basewidth,basedepth - ширина и глубина основания
apexheight - высота апекса (вершины, противолежащей основанию)
cutheight - высота отсечения
parent - id родителя для усеченной пирамиды (0 - отсутствие родителя).

DodecahedronCreate

real = DodecahedronCreate ( parent as real);
Создает додекаэдр (многогранник, составленный из 12 правильных пятиугольников) и возвращает его id.
parent - id родителя для додекаэдра (0 - отсутствие родителя).

IcosahedronCreate

real = IcosahedronCreate ( parent as real);
Создает икосаэдр (многогранник, составленный из 20 равносторонних треугольников) и возвращает его id.
parent - id родителя для икосаэдра (0 - отсутствие родителя).

TeapotCreate

real = TeapotCreate ( parent as real);
Создает чайник Юта и возвращает его id.
