Изменения API


Xtreme3D 3.х не является на 100% обратно совместимым с веткой 2.х - есть несколько важных изменений, но в целом имена функций и их смысл не поменялись. Реализована практически вся старая функциональность из Xtreme3D 2.0, за исключением:

- опрашивания позиции и нормали последнего столкновения в DCE и FPSManager
- опрашивания количества отрендеренных треугольников
- portal culling'а. Как он работал (и работал ли вообще) в прежней версии движка, непонятно, поэтому эта функциональность не была реализована.

Функции, которых больше нет в Xtreme3D 3.x:
- LoadB3D (функция работала не для всех файлов формата B3D, сейчас этот формат загружается обычным способом, через FreeformCreate)
- ClearZones, ZonesVisible, CullZones, ZoneAddPortal, ZoneObjectInZone, ZoneGetCurrent, ZoneGetNew, ZoneAdjustSize, CreateZones, ZoneGetPortalCount, ZoneGetPortal
- TextSetColor, TextSetFont, TextSetText (появились соответствующие функции для каждого типа текста)
- FreeformSetSecondTextureCoords, MeshOptimize, MeshSmoothFaces
- ObjectAlignToCamera, ObjectGetAtXY, ObjectSetDiffuseColor, ObjectStructureChanged, ObjectClueToBone
- ObjectIsClipped (функция обратна ObjectInFrustrum по своему назначению)
- MaterialGetFromLib
- MaterialAddTextureEx, MaterialTextureExClear, MaterialTextureExDelete (система TextureEx была переписана для совместимости с шейдерами, и API был изменен)
- ThorFXSetTargetAsObject
- DceGetGravity, DceVelocityCollided, DceGravityCollided, DceCountCollisions, DceGetCollidedObject, DceGetCollisionPosition, DceGetCollisionNormal
- FpsCountCollisions, FpsClearCollisions, FpsGetCollisionPosition, FpsGetCollisionNormal, FpsGetCollidedObject

Мы заинтересованы в том, чтобы вернуть некоторые из этих функций в движок, поэтому, если у вас есть соответствующий опыт, знания и желание нам помочь, просим связаться по почте (gecko0307@gmail.com) или через форум: https://xtreme3d.ru/forum.
